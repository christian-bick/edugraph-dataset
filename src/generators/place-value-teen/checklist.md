### Place Value Teen Visual Checklist

- The problem represents a number in the teen range (11 to 19).
- For Compose Teen:
  - The problem must show a "10" box and a separate "ones" box (e.g. 5 ones).
  - The solution must highlight the button showing the correct total teen number (e.g. 15).
- For Decompose Teen:
  - The problem must show a total teen number (e.g. 15).
  - The solution must highlight the correct decomposition into "10" and the remaining "ones" (e.g. 5 ones).
