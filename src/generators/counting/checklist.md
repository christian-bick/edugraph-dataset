### Counting Visual Checklist
- **Object Visibility:** All objects must be clearly visible and not overlapping each other.
- **Horizontal Flow:** For `counting-inc-dec`, the arrow and number boxes must be aligned horizontally.
- **Icon Consistency:** The same icon must be used for all objects in a single problem (but varies between problems).
- **Triangle '1':** The increment/decrement triangles must contain a bold, white "1" clearly visible inside.
