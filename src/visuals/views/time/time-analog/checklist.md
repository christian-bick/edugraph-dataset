# Time Analog - Visual & Rendering Checklist

## Clock Face
- The clock face is a circle with 12 tick marks: a larger dot at each 5-minute position and a smaller dot at each remaining minute position. No numeric hour labels ("1", "2", ... "12" as text) are ever rendered — tick marks are the complete, intended design, not a missing element.

## Two Valid Exercise Layouts
Exactly one of the following two layouts is correct per image — never a mix of both, and never neither:
- **Reading layout:** Clock hands are visible in both Question Mode and Solution Mode. A boxed digital time readout is present but empty/blank in Question Mode, and only appears (in green, highlighted) in Solution Mode.
- **Drawing layout:** A boxed digital time readout with a dashed border is visible in both Question Mode and Solution Mode, showing the target time. Clock hands are completely absent in Question Mode — this is intentional and correct, not a rendering failure, since the exercise asks the student to determine the hand positions — and appear in green only in Solution Mode.

## Prompt
- Per the global Instruction & Mode Rules, a prompt is shown only in Question Mode and never in Solution Mode: "What time is it?" for the Reading layout, "What time should the clock show?" for the Drawing layout.
