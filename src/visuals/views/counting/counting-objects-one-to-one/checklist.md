### Counting Objects One-to-One View Checklist

#### Visual Layout & Rendering Requirements
1. **Instruction:**
   - In Question Mode, the question text "Count the objects and write the numbers in order." must be clearly visible above the canvas. Per the global Instruction & Mode Rules, this text must be absent in Solution Mode.
2. **One-to-One Number Tagging:**
   - In **Question Mode (`_mode-Q`)**, each object must display a small, blank bubble underneath it.
   - In **Solution Mode (`_mode-S`)**, each bubble must be filled with a sequential number (1, 2, ..., `numObjects`) highlighted in green.
3. **Canvas & Answer Box:**
   - Exactly `numObjects` count of objects must be generated and rendered.
   - Answer box at the bottom must be blank in Question Mode, and filled with the total count (`numObjects`) in Solution Mode.
